# Tools & Integrations

Reference for connected services and API keys. Update this when a new tool is connected or a behavior note is worth keeping.

## Built-in tools

- **Memory** — `add_memory` (long_term / skill), `update_long_term_memory`
- **User profile** — `update_user_profile`, `update_soul`
- **Heartbeat** — `update_heartbeat`, read heartbeat schedule
- **Files** — read, create, list files and directories on the server
- **Shell** — run shell commands on the server
- **Code** — run JavaScript/Python code in a sandbox
- **Fetch** — fetch URLs and web pages
- **Buttons** — send inline Telegram buttons
- **Reactions** — send emoji reactions to messages
- **Skills** — `use_skill` to switch active skill
- **Subagent** — `spawn_agent` for isolated one-shot queries
